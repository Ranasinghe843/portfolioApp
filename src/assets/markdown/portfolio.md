# Professional Portfolio Web App – Developer and Designer

**Personal Project | [Live Site](https://ranasinghe843.github.io/portfolioApp/)**

### Project Overview

I engineered a high-performance single-page application (SPA) to serve as a centralized, interactive hub for my engineering projects, research, and professional background. The application is designed to bridge the gap between complex engineering data and a polished, user-centric frontend experience.

  * **Role:** Frontend Developer and UI Designer.
  * **Tech Stack:** Angular (TypeScript), Tailwind CSS, HTML5, GitHub Pages.
  * **Key Focus:** Data-driven architecture, modular component design, and interactive UX/UI.

-----

### Data-Driven Architecture and Dynamic Loading

To ensure the portfolio is easily maintainable and scalable, I implemented a decoupled data architecture.

  * **JSON-Based Data Store:** Centralized all project, experience, and education details into a data.json file, allowing for content updates without modifying the application logic.
  * **Dynamic Component Rendering:** Engineered an Angular Service to fetch and parse the JSON data, dynamically injecting it into TypeScript-based components.
  * **TypeScript Interfacing:** Defined strict interfaces for all data objects to ensure type safety and consistent rendering across the platform.

-----

### UI/UX and Advanced Interactivity

The visual identity of the site combines a minimalist professional aesthetic with high levels of interactivity using Tailwind CSS.

  * **Fluid Hover Animations:** Implemented custom hover state transitions across all interactive elements, including navigation links, social icons, resume buttons, and project URLs.
  * **Interactive Project Cards:** Developed specialized project and experience cards that utilize hover triggers to reveal linked documentation or external project links.
  * **Responsive Utility-First Styling:** Leveraged Tailwind's utility classes to build a fully responsive grid system that maintains structural integrity from mobile devices to desktop workstations.

-----

### Performance and Deployment

The application is optimized for fast load times and reliable delivery through a modern development pipeline.

  * **Optimized Asset Delivery:** Configured the Angular build pipeline to minify production code and optimize the delivery of the centralized JSON data.
  * **GitHub Pages Hosting:** Deployed the application as a static site, ensuring high availability and seamless integration with existing source code repositories.

-----

### Key Features at a Glance

  * **Dynamic Project Gallery:** Interactive cards that highlight key technologies used in each engineering project.
  * **Unified Navigation:** A persistent, animated navigation bar providing a seamless transition between sections.
  * **Resource Integration:** Direct, animated pathways to professional assets like my latest resume and GitHub repositories.

> **Technical Expertise Applied:** Angular, TypeScript, Tailwind CSS, JSON Data Management, Responsive Web Design, Single-Page Application (SPA) Logic.
